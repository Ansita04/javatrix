Introduction to Java

Java is a high-level, class-based, object-oriented programming language that is widely used for building applications. and is designed to have as few implementation dependencies as possible.
It was developed by James Gosling at Sun Microsystems and released in 1995. Java applications are compiled into bytecode that runs on the Java Virtual Machine (JVM), making it platform-independent.

History

Java was developed by James Gosling at Sun Microsystems in 1995. It is now maintained by Oracle Corporation.

Basic Syntax

The following code demonstrates a simple Java program that prints "Hello, World!" to the console.

            // This is a simple Java program
            public class HelloWorld {
                public static void main(String[] args) {
                    // Prints "Hello, World!" to the console
                    System.out.println("Hello, World!");
                }
            }

Variables and Data Types

Java has different types of variables such as:

  • int: Integer values (e.g., 10, 20, 30)
  • double: Decimal values (e.g., 10.5, 20.75)
  • char: Single character (e.g., 'A', 'B')
  • boolean: True or false values

Rules for Declaring Variables

  • Variable names must start with a letter, underscore (_), or dollar sign ($).
  • Variable names cannot be Java reserved keywords (e.g., class, public).
  • Variable names are case-sensitive (e.g., age and Age are different).
  • Use meaningful names that represent the value being stored.
  • Constants (unchangeable variables) should be written in uppercase (e.g., final int MAX_VALUE = 100;).

Example: Declaring Variables

            // Declaring variables
            int age = 25;
            double price = 99.99;
            char grade = 'A';
            boolean isJavaFun = true;
