Object-Oriented Programming (OOP) in Java

Java follows the object-oriented programming paradigm, which is based on the concepts of objects and classes.

Key OOP Principles

  • Encapsulation: Wrapping data and methods into a single unit (class) to protect it from unauthorized access.
  • Inheritance: Enabling a class to inherit properties and methods from another class, reducing code redundancy.
  • Polymorphism: Allowing methods to perform different behaviors based on the object, enhancing flexibility.
  • Abstraction: Hiding implementation details and exposing only necessary parts to reduce complexity.

Example: Class and Object

The following example demonstrates class creation, object instantiation, and method usage.

        class Car {
            String model;
            int year;
            
            // Constructor to initialize values
            Car(String model, int year) {
                this.model = model;
                this.year = year;
            }
            
            // Method to display car details
            void display() {
                System.out.println("Car Model: " + model + ", Year: " + year);
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                Car myCar = new Car("Toyota", 2022);
                myCar.display();
            }
        }

Encapsulation Example

Encapsulation restricts direct access to class fields by making them private and provides getter and setter methods.

        class Person {
            private String name; // Private variable (data hiding)
            
            // Getter method
            public String getName() {
                return name;
            }
            
            // Setter method
            public void setName(String newName) {
                this.name = newName;
            }
        }

Polymorphism Example

Polymorphism allows the same method to perform different tasks.

        class Animal {
            void makeSound() {
                System.out.println("Animal makes a sound");
            }
        }
        
        class Dog extends Animal {
            @Override
            void makeSound() {
                System.out.println("Dog barks");
            }
        }

Abstraction Example

Abstraction is implemented using abstract classes or interfaces.

        abstract class Vehicle {
            abstract void start(); // Abstract method
        }
        
        class Car extends Vehicle {
            void start() {
                System.out.println("Car starts with a key");
            }
        }
